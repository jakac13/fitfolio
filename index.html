Fitfolio.

The digital wardrobe mobile app provides a convenient way to keep track of your entire clothing collection by storing photos and details of each item. This allows you to quickly and easily create outfits and avoid outfit repetition.

MVP

User can login using a Google account.

User can search between the categories and use the search bar.

User can add in new clothes. Images can be taken with a camera or picked from the gallery.

User can delete items from his digital closet.

User can log out and log in with another Google account.
